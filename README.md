# OPD_Python_Test
Набор юнит тестов для вебсервера на Flask, созданного в рамках ЛР3.
Корректный запуск тестов подразумевает заранее включённый сервер (в отдельном терминале)

- main.py - Запуск сервера
- test.py - Запуск юнит тестов
